new 绑定

介绍

new 操作符是 JavaScript 中用于创建对象实例的关键字。当使用 new 操作符时，它会执行以下几个步骤：

  1. 创建一个空的普通 JavaScript 对象（简称为 instance），继承自构造函数的原型对象（prototype）。
  2. 将构造函数的作用域（this）绑定到新创建的实例对象上，以便在构造函数中使用 this 来引用这个实例。
  3. 执行构造函数中的代码，初始化实例的属性和方法。
  4. 如果构造函数没有显式返回一个对象，则返回新创建的实例对象。

call

AIGC

call 是 JavaScript 中的一个函数方法，它允许你在一个对象上调用另一个对象的方法，并且可以显式指定该方法的上下文（即函数内部的this指向）。call的基本语法如下：

  • function: 要调用的函数。
  • thisArg: 函数被调用时的上下文，即函数内部的this指向。
  • arg1, arg2, ...: 传递给函数的参数。

call 方法会临时改变函数的执行上下文，然后立即调用函数，并返回函数执行的结果。

bind

bind 会返回一个新的函数, 该函数的 this 指向 bind 的第一个参数

AIGC

在JavaScript中，bind是一个内置方法，用于创建一个新的函数，并将其上下文（也就是函数内部的this关键字）绑定到指定的对象。绑定后的函数可以在后续调用中保持绑定的上下文，无论在什么环境下调用，this都会指向绑定的对象。

bind方法的语法如下：

  • thisArg: 要绑定到函数上的对象，即将函数执行时的this关键字绑定到这个对象。
  • arg1, arg2, ...: 传递给函数的参数，可以在调用时被传入函数。
