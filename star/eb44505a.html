前端资源

前端技术学习和文档

  • MDN Web Docs: Mozilla 开发者网络提供的全面的 Web 技术文档和教程。
  • W3Schools: Web 开发入门教程和参考手册。
  • DevDocs: 聚合了各种前端技术的在线文档，包括 HTML、CSS、JavaScript 等。
  • CSS-Tricks: 提供关于 CSS、HTML 和 JavaScript 的实用技巧和教程。
  • freeCodeCamp: 提供免费的编码课程，涵盖前端和后端开发技术。
  • promiseA+: promiseA+

在线学习平台

  • Codecademy: 提供互动式编程课程，涵盖前端和后端技术。
  • Udemy: 提供广泛的在线编程课程，包括前端开发。
  • Coursera: 合作高校的在线课程平台，提供前端开发相关的课程。
  • Pluralsight: 提供技术相关的视频课程，包括前端开发。

前端社区和论坛

  • Stack Overflow: 一个广泛使用的程序员问答社区，提供前端开发相关问题的解答。
  • GitHub: 世界上最大的代码托管平台，你可以在上面找到开源前端项目和参与贡献。
  • CodePen: 在线代码编辑器和社交编程环境，可以分享和发现前端代码片段。
  • DEV Community: 开发者社区，包含许多前端开发相关的文章和讨论。

前端工具和资源

  • npm: Node.js 包管理器，提供了许多前端包和库。
  • Webpack: 前端模块打包工具，用于构建现代化的 Web 应用程序。
  • Can I use: 查看不同 Web 技术在不同浏览器的兼容性情况。
  • Unsplash: 免费高质量的照片资源，可用于网站背景和设计。

前端新闻和博客

  • CSS-Tricks: 前端技巧、教程和文章。
  • Smashing Magazine: 设计和开发相关的文章和资源。
  • Hacker News: 科技领域的新闻和讨论，包括前端技术。

请注意，网站和资源可能会随时间而变化，所以建议定期检查这些网站以确保获取最新信息。
