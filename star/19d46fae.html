VS Code快捷键

Human:
总结一些使用vscode前端开发操作代码的快捷键

Assistant:

下面是一些常用的 VS Code 前端开发快捷键：

  1. Ctrl + Shift + P：打开命令面板
  2. Ctrl + Shift + E：打开/关闭侧边栏文件浏览器
  3. Ctrl + Shift + F：全局搜索
  4. Ctrl + F：当前文件内搜索
  5. Ctrl + Shift + K：删除当前行
  6. Alt + ↑/↓：上下移动当前行
  7. Ctrl + D：选中下一个相同的单词
  8. Ctrl + /：注释/取消注释当前行或选中区域
  9. Ctrl + Shift + L：选中所有相同的单词
  10. Ctrl + Shift + I：格式化选中区域
  11. Ctrl + Shift + V：在侧边栏打开 Markdown 预览
  12. Ctrl + Shift + X：打开扩展面板
  13. Ctrl + Shift + H：替换所有匹配项
  14. Alt + Z：切换单词换行模式

当然，这只是一小部分常用的快捷键，你可以通过按下 Ctrl + K Ctrl + S 打开 VS Code 快捷键提示窗口，浏览并自定义你的快捷键。

Exported on 2023年4月18日 下午11:51.

自我总结

  1. ALT+↑↓ 快速移动代码行
  2. shift+ALT+↑↓ 快速复制代码
  3. Ctrl+/ 单行注释
  4. ALT+shift+A 块级注释
  5. Ctrl+L 选中当前一整行
  6. Ctrl+I 代码提示
  7. /**+回车 函数参数注释
  8. F12 查看函数定义
